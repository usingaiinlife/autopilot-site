
GitHub Pages Pack (No Netlify)
------------------------------

Deploy:
1) Create repo (usingaiinlife.github.io for a user site, or any repo for a project site).
2) Upload all files and folders in this 'site' directory to the repo root.
3) Settings → Pages → Source: Deploy from a branch → Branch: main → Folder: /(root) → Save.

Customize:
- Formspree (contact.html): replace ACTION_FORMSPREE with your endpoint.
- Gumroad links (products/*.html): replace GUMROAD_URL_HERE.
- Optional AI chat: set window.BOTPRESS_BOT_ID in assets/js/main.js.
